Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element defines the title of the webpage to help with search engine optimization.
  • For the body element, h1 is the main heading and the largest; options go through h6 getting smaller each time.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • An external css style sheet must be linked to the HTML file.
  • The * symbol is a wild card that applies the style to the whole web page.

Git

  • ls: lists what files or folders are present in current directory
  • cd: change directory, go into folder
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: receives any branch modifications to your local environment from the origin branch specified
